## 答え

- htmlの答えは `answer.html` に記述しています。(answer.htmlをブラウザで開くと正解の挙動を確認できます。)
- jsの答えは `answer.js` に記述しています。
- 全く作業が進められなくなったときに答えを参考にしても良いですが、コピー&ペーストは厳禁です
